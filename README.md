# Kaif Islam - Premium Portfolio

Next.js App Router portfolio that combines AI/ML engineering and full-stack development.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Deploy (Vercel)

1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Framework preset: `Next.js`.
4. Build command: `npm run build`.
5. Output: default (`.next` managed by Vercel).

Update `metadataBase`, `sitemap.ts`, and `robots.ts` with your live domain after deployment.
